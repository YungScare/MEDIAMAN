# MEDIAMAN

Проект верстки для MEDIAMAN с поддержкой desktop и mobile версий.

## Структура проекта

```
mediaman_v1/
├── index.html          # Главная страница
├── css/
│   └── style.css       # Основные стили с медиа-запросами
└── README.md
```

## Технологии

- HTML5
- CSS3 (с медиа-запросами для адаптивности)

## Адаптивность

Проект поддерживает три основных разрешения:
- **Desktop**: от 1024px и выше
- **Tablet**: от 768px до 1023px
- **Mobile**: до 767px
